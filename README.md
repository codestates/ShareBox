## About
> 마트에서 파는 대용량의 음식,<br>
> 양이 많아 구입이 망설여지셨나요?<br>
> 대용량의 음식을 나눌 파티원을 구할 수 있습니다.

## Feature
> * 대용량의 음식을 검색을 통해서 원하는 카테고리의 음식만 이웃과 나눌 수 있습니다.
> * 게시글을 통해 이 음식의 대한 평가를 볼 수 있습니다.
> * 댓글을 확인하지 않아도 게시글의 나눔완료 여부를 확인할 수 있습니다.
